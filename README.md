#  D2 Dashboard — Destiny 2 Activity Tracker

Este projeto é um dashboard interativo desenvolvido em Python para visualizar estatísticas de atividades do jogo **Destiny 2**, como modos PvE, Raids e Masmorras.

Os dados são carregados diretamente de planilhas online (Google Sheets) e atualizados em tempo real.

## Funcionalidades

-  Visualização de dados em gráficos interativos
-  Filtros por modo, raid e masmorra
-  Ordenação por tempo (mais rápido / média)
-  Atualização automática via planilha na nuvem
-  Publicação online com Streamlit

##  Tecnologias Utilizadas

- Python
- Streamlit
- Pandas
- Plotly
- Google Sheets
